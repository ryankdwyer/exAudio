# exAudio

A cross platform audio player built in Electron. exAudio is capable of storing and playing a library of various audio files in an array of codecs, including:

1. flac
2. alac
3. aac 
4. mp3
5. mp4
6. more are coming!

##Technologies/Libraries used:

1. Electron
2. AngularJS
3. aurorajs
4. flac.js
5. mp3.js
6. aac.js
7. alac.js
8. lokijs
9. d3
10. fft-js
